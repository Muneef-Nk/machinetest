- Flutter 3.41.2

## 🏗 Architecture
- **Provider** for state management  


## 💡 Features
- User login (with static token)  
- Home banners and category list  
- Product details API integration  
- Cart management (in-memory with add/remove/quantity)  
- Snackbars for cart actions
